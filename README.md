# VitalityHub – Health & Sport Website

## Overview
VitalityHub is an interactive web platform designed to promote health and sport awareness. The site provides articles, daily fitness tips, and a feedback system to engage users in their wellness journey.

## Features
- **Responsive Design**: Adapts to all screen sizes.
- **Interactive Elements**: 
  - Daily health tip generator (AJAX-powered)
  - Smooth navigation buttons
  - Feedback form with live validation
- **Engaging Content**: Articles on exercise, nutrition, and mental health.
- **Visual Appeal**: Modern gradient design with cards and shadows.

## Technical Stack
- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **Deployment**: Render (Python 3 runtime)

## How to Run Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open `http://localhost:5000` in your browser.

## Project Glossary Words Used
1. **Interactive** – The page includes buttons and AJAX calls.
2. **Prototype** – This is a working website prototype.
3. **Engaging** – Design aims to engage users visually and functionally.
4. **Layout** – The page layout is organized into clear sections.
5. **Visual** – Strong visual design with colors, gradients, and icons.
